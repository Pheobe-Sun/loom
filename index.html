Loom

A library by Cora Johnson-Roberson for the Processing programming environment.
Last update, 10/09/2016.

Patterns that change over time, flexibly mapped to audiovisual output.
Sequences of events can be rendered both sonically (through MIDI, Minim, SuperCollider, etc.) and graphically (through Processing itself).

Download

Download Loom version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted loom folder into the libraries folder of your Processing sketches. Reference and examples are included in the loom folder.

Keywords. patterns, timing

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.

Source. The source code of Loom is available at Github, and its repository can be browsed here.
